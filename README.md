# Simpele Website Starter (GitHub Pages)

Dit is een minimale, statische website (alleen HTML/CSS/JS) die automatisch publiceert via **GitHub Pages** met **GitHub Actions**.

## Snelstart
1. Maak op GitHub een nieuwe repository aan (privé of publiek).
2. Upload *alle* bestanden uit deze map naar die repo (of push via VS Code).
3. Ga in je repo naar **Settings → Pages** en zet **Build and deployment** op **GitHub Actions** (éénmalig).
4. Doe een eerste commit/push naar `main`. De workflow bouwt en publiceert automatisch.
5. Jouw site komt online op: `https://<username>.github.io/<REPO_NAME>/`

## Lokaal bekijken
Open `index.html` direct in je browser, of installeer in VS Code de extensie **Live Server** en kies *Open with Live Server*.
